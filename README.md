# algoholic.js
==================

Classic algorithms and data structures implemented in JavaScript. Build by taha Nebti for acm tcpc training.


### Installing
```
npm install --save algoholic
```

### Contents

#### Math algorithms

```javascript
require('algorithms/math');
// or
require('algorithms').Math;
```
* collatzConjecture
* extendedEuclidean
* fastPower
* fibonacci
* findDivisors
* fisherYates
* gcd (Greatest common divisor)
* greatestDifference
* lcm (Least common multiple)
* newtonSqrt
* nextPermutation
* powerSet
* reservoirSampling
* shannonEntropy
